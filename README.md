# 🚀 AI Resume Screener

An AI-powered system that helps HR teams automatically screen and rank resumes based on a job description.

---

## 🎯 Problem

HR teams receive many resumes and manually reviewing them is:
- Time-consuming  
- Inconsistent  
- Inefficient  

---

## 💡 Solution

This project uses AI to:
- Analyze resumes against a job description  
- Score each candidate (0–100)  
- Provide reasoning  
- Rank candidates automatically  

---

## ⚙️ Features

- 📄 Upload Job Description (PDF)
- 👥 Upload multiple resumes
- 🤖 AI scoring
- 📊 Ranked output
- 🧠 Reason for each score

---

## 🏗️ Tech Stack

- Frontend: React + Vite  
- Backend: Node.js + Express  
- AI: Groq (LLaMA 3)  
- PDF Processing: pdfjs  

---

## 🔄 How It Works

1. Upload Job Description  
2. Upload resumes  
3. Backend extracts text  
4. AI compares resumes  
5. Returns ranked results  

---

## 🚀 Run Locally

### 1. Clone
```bash
git clone https://github.com/sivasubrahmanyam139-commits/resume-screener.git
cd resume-screener
```

### 2. Backend
```bash
cd backend
npm install
node index.js
```

### 3. Frontend (new terminal)
```bash
cd frontend
npm install
npm run dev
```

### 4. Open
http://localhost:5173/
